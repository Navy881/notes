# Event sourcing (–ü–æ—Ä–æ–∂–¥–µ–Ω–∏–µ —Å–æ–±—ã—Ç–∏–π)

# **–ü—Ä–æ–±–ª–µ–º–∞**

–ö–∞–∫ –∞—Ç–æ–º–∞—Ä–Ω–æ –æ–±–Ω–æ–≤–∏—Ç—å –±–∞–∑—É –¥–∞–Ω–Ω—ã—Ö –∏ –æ—Ç–ø—Ä–∞–≤–∏—Ç—å —Å–æ–æ–±—â–µ–Ω–∏—è –±—Ä–æ–∫–µ—Ä—É —Å–æ–æ–±—â–µ–Ω–∏–π?

–¢—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω—ã–π –ø–æ–¥—Ö–æ–¥ –∫ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ –ø–æ–¥—Ä–∞–∑—É–º–µ–≤–∞–µ—Ç –ø—Ä–∏–≤—è–∑—ã–≤–∞–Ω–∏–µ –∫–ª–∞—Å—Å–æ–≤ –∫ —Ç–∞–±–ª–∏—Ü–∞–º –±–∞–∑ –¥–∞–Ω–Ω—ã—Ö, –ø–æ–ª–µ–π —ç—Ç–∏—Ö –∫–ª–∞—Å—Å–æ–≤ ‚Äî –∫ —Å—Ç–æ–ª–±—Ü–∞–º, –∞ –∏—Ö —ç–∫–∑–µ–º–ø–ª—è¬≠ —Ä–æ–≤ ‚Äî –∫ —Å—Ç—Ä–æ–∫–∞–º —ç—Ç–∏—Ö —Ç–∞–±–ª–∏—Ü.

–£ —Ç—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞ –µ—Å—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –Ω–µ–¥–æ—Å—Ç–∞—Ç–∫–æ–≤ –∏ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–π.

- –û–±—ä–µ–∫—Ç–Ω–æ-—Ä–µ–ª—è—Ü–∏–æ–Ω–Ω—ã–π —Ä–∞–∑—Ä—ã–≤. –ó–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–æ–º, –∫–æ–Ω—Ü–µ–ø—Ç—É–∞–ª—å–Ω–æ–º –Ω–µ—Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–∏ –º–µ–∂–¥—É —Ç–∞–±–ª–∏—á–Ω–æ–π —Ä–µ–ª—è—Ü–∏–æ–Ω–Ω–æ–π —Å—Ö–µ–º–æ–π –∏ –≥—Ä–∞—Ñ–æ–≤–æ–π —Å—Ç—Ä—É–∫—Ç—É—Ä–æ–π —Ä–∞–∑–≤–∏—Ç–æ–π –¥–æ–º–µ–Ω–Ω–æ–π –º–æ¬≠–¥–µ–ª–∏ —Å –µ–µ —Å–ª–æ–∂–Ω—ã–º–∏ –æ—Ç–Ω–æ—à–µ–Ω–∏—è–º–∏.
- –û—Ç—Å—É—Ç—Å—Ç–≤–∏–µ –∏—Å—Ç–æ—Ä–∏–∏ –∞–≥—Ä–µ–≥–∞—Ç–æ–≤. –ï—â–µ –æ–¥–Ω–æ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ —Ç—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞ –∫ —Ö—Ä–∞–Ω–µ–Ω–∏—é –¥–∞–Ω–Ω—ã—Ö —Å–æ—Å—Ç–æ–∏—Ç –≤ —Ç–æ–º, —á—Ç–æ –≤ –Ω–µ–º –ø—Ä–µ–¥—É—Å–º–æ—Ç—Ä–µ–Ω–æ —Ö—Ä–∞–Ω–µ–Ω–∏–µ –ª–∏—à—å —Ç–µ–∫—É—â–µ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è –∞–≥—Ä–µ–≥–∞—Ç–∞. –ü–æ—Å–ª–µ –æ–±–Ω–æ–≤–ª–µ–Ω–∏—è –∞–≥—Ä–µ–≥–∞—Ç–∞ –µ–≥–æ –ø—Ä–µ–¥—ã–¥—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ç–µ—Ä—è–µ—Ç—Å—è.
- –†–µ–∞–ª–∏–∑–∞—Ü–∏—è –∂—É—Ä–Ω–∞–ª–∏—Ä–æ–≤–∞–Ω–∏—è –¥–ª—è –∞—É–¥–∏—Ç–∞ —Ç—Ä–µ–±—É–µ—Ç –º–Ω–æ–≥–æ —É—Å–∏–ª–∏–π –∏ —á—Ä–µ–≤–∞—Ç–∞ –æ—à–∏–±–∫–∞–º–∏.
- –ü—É–±–ª–∏–∫–∞—Ü–∏—è —Å–æ–±—ã—Ç–∏–π —è–≤–ª—è–µ—Ç—Å—è –ª–∏—à—å –¥–æ–ø–æ–ª–Ω–µ–Ω–∏–µ–º –∫ –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫–µ. –ï—â–µ –æ–¥–Ω–æ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ —Ç—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞ –∫ —Ö—Ä–∞–Ω–µ–Ω–∏—é –¥–∞–Ω–Ω—ã—Ö —Å–æ—Å—Ç–æ–∏—Ç –≤ —Ç–æ–º, —á—Ç–æ –æ–Ω –æ–±—ã—á–Ω–æ –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –ø—É–±–ª–∏–∫–∞—Ü–∏—é –¥–æ–º–µ–Ω–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π. –†–∞–∑—Ä–∞¬≠–±–æ—Ç—á–∏–∫–∞–º –ø—Ä–∏—Ö–æ–¥–∏—Ç—Å—è –¥–æ–ø–∏—Å—ã–≤–∞—Ç—å –ª–æ–≥–∏–∫—É –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ —Å–æ–±—ã—Ç–∏–π, –∫–æ—Ç–æ—Ä–∞—è –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω–æ –º–æ–∂–µ—Ç —É—Ç—Ä–∞—Ç–∏—Ç—å —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏—é —Å –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫–æ–π.

# –†–µ—à–µ–Ω–∏–µ

–ü–æ—Ä–æ–∂–¥–µ–Ω–∏–µ —Å–æ–±—ã—Ç–∏–π ‚Äî —ç—Ç–æ –µ—â–µ –æ–¥–∏–Ω —Å–ø–æ—Å–æ–± —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–∏—è –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫–∏ –∏ —Å–æ¬≠ —Ö—Ä–∞–Ω–µ–Ω–∏—è –∞–≥—Ä–µ–≥–∞—Ç–æ–≤. –ê–≥—Ä–µ–≥–∞—Ç—ã —Å–æ—Ö—Ä–∞–Ω—è—é—Ç—Å—è –≤ –≤–∏–¥–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ —Å–æ–±—ã—Ç–∏–π, –∫–∞–∂–¥–æ–µ –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç –∏–∑–º–µ–Ω–µ–Ω–∏–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è –∞–≥—Ä–µ–≥–∞—Ç–∞. –ü—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –≤–æ—Å¬≠ —Å–æ–∑–¥–∞–µ—Ç —Ç–µ–∫—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ, –≤–æ—Å–ø—Ä–æ–∏–∑–≤–æ–¥—è –∑–∞–ø–∏—Å–∞–Ω–Ω—ã–µ —Å–æ–±—ã—Ç–∏—è.

## **–°–æ—Ö—Ä–∞–Ω–µ–Ω–∏–µ –∞–≥—Ä–µ–≥–∞—Ç–æ–≤ —Å –ø–æ–º–æ—â—å—é –ø–æ—Ä–æ–∂–¥–µ–Ω–∏—è —Å–æ–±—ã—Ç–∏–π**

–ü–æ—Ä–æ–∂¬≠–¥–µ–Ω–∏–µ —Å–æ–±—ã—Ç–∏–π –æ—Å–Ω–æ–≤–∞–Ω–æ –Ω–∞ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏ –¥–æ–º–µ–Ω–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π –∏ —Ä–∞–±–æ—Ç–∞–µ—Ç —Å–æ–≤—Å–µ–º –∏–Ω–∞—á–µ. –û–Ω–æ —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç –∫–∞–∂–¥—ã–π –∞–≥—Ä–µ–≥–∞—Ç –≤ –±–∞–∑–µ –¥–∞–Ω–Ω—ã—Ö, —Ç–∞–∫ –Ω–∞–∑—ã–≤–∞–µ–º–æ–º —Ö—Ä–∞–Ω–∏–ª–∏—â–µ —Å–æ–±—ã—Ç–∏–π, –≤ –≤–∏–¥–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ —Å–æ–±—ã—Ç–∏–π. –í–æ–∑—å–º–µ–º –≤ –∫–∞—á–µ—Å—Ç–≤–µ –ø—Ä–∏–º–µ—Ä–∞ –∞–≥—Ä–µ–≥–∞—Ç Order. –í–º–µ—Å—Ç–æ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è –∫–∞–∂–¥–æ–≥–æ —ç–∫–∑–µ–º¬≠–ø–ª—è—Ä–∞ Order –≤ –≤–∏–¥–µ —Å—Ç—Ä–æ–∫–∏ —Ç–∞–±–ª–∏—Ü—ã ORDER –ø–æ—Ä–æ–∂–¥–µ–Ω–∏–µ —Å–æ–±—ã—Ç–∏–π –ø–æ–º–µ—â–∞–µ—Ç –∫–∞–∂–¥—ã–π –∞–≥—Ä–µ–≥–∞—Ç Order –≤ —Ç–∞–±–ª–∏—Ü—É EVENTS, –∏—Å–ø–æ–ª—å–∑—É—è –æ–¥–Ω—É –∏–ª–∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å—Ç—Ä–æ–∫ (—Ä–∏—Å. 6.2). –ö–∞–∂–¥–∞—è —Å—Ç—Ä–æ–∫–∞ ‚Äî —ç—Ç–æ –¥–æ–º–µ–Ω–Ω–æ–µ —Å–æ–±—ã—Ç–∏–µ, —Ç–∞–∫–æ–µ –∫–∞–∫ Order Created, Order Approved, Order Shipped –∏ —Ç. –¥.

–°–æ–∑–¥–∞–≤–∞—è –∏–ª–∏ –æ–±–Ω–æ–≤–ª—è—è –∞–≥—Ä–µ–≥–∞—Ç, –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –≤—Å—Ç–∞–≤–ª—è–µ—Ç –≤ —Ç–∞–±–ª–∏—Ü—É EVENTS —Å–æ–±—ã—Ç–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ —Ç–æ—Ç —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞–ª. –ü—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –∑–∞–≥—Ä—É–∂–∞–µ—Ç –∞–≥—Ä–µ–≥–∞—Ç –∏–∑ —Ö—Ä–∞–Ω–∏–ª–∏—â–∞, –∏–∑–≤–ª–µ–∫–∞—è –∏ –≤–æ—Å–ø—Ä–æ–∏–∑–≤–æ–¥—è —Å–æ–±—ã—Ç–∏—è. –ï—Å–ª–∏ –≥–æ–≤–æ—Ä–∏—Ç—å –±–æ–ª–µ–µ –ø–æ–¥—Ä–æ–±–Ω–æ, –∑–∞–≥—Ä—É–∑–∫–∞ –∞–≥—Ä–µ–≥–∞—Ç–∞ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —Å–ª–µ–¥—É—é—â–∏—Ö —Ç—Ä–µ—Ö —à–∞–≥–æ–≤.

1. –ó–∞–≥—Ä—É–∑–∫–∞ —Å–æ–±—ã—Ç–∏–π –∞–≥—Ä–µ–≥–∞—Ç–∞.
2. –°–æ–∑–¥–∞–Ω–∏–µ —ç–∫–∑–µ–º–ø–ª—è—Ä–∞ –∞–≥—Ä–µ–≥–∞—Ç–∞ —Å –ø–æ–º–æ—â—å—é –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–∞ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é.
3. –ü–µ—Ä–µ–±–æ—Ä —Å–æ–±—ã—Ç–∏–π —Å –≤—ã–∑–æ–≤–æ–º apply().

### **–°–æ–±—ã—Ç–∏—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç –∏–∑–º–µ–Ω–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏—è**

–ü—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω–Ω–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞ —Å–æ–±—ã—Ç–∏—è –≥–µ–Ω–µ—Ä–∏—Ä—É—é—Ç—Å—è –≤—Å–µ–≥–¥–∞. –ö–∞–∂–¥–æ–µ –∏–∑–º–µ–Ω–µ–Ω–∏–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è –∞–≥—Ä–µ–≥–∞—Ç–∞, –≤–∫–ª—é—á–∞—è –µ–≥–æ —Å–æ–∑–¥–∞–Ω–∏–µ, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–æ –¥–æ–º–µ–Ω¬≠–Ω—ã–º —Å–æ–±—ã—Ç–∏–µ–º. –ö–∞–∂–¥—ã–π —Ä–∞–∑, –∫–æ–≥–¥–∞ –∞–≥—Ä–µ–≥–∞—Ç –º–µ–Ω—è–µ—Ç —Å–≤–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ, –æ–Ω –æ–±—è–∑–∞–Ω —Å–≥–µ–Ω–µ¬≠—Ä–∏—Ä–æ–≤–∞—Ç—å —Å–æ–±—ã—Ç–∏–µ. –ù–∞–ø—Ä–∏–º–µ—Ä, –∞–≥—Ä–µ–≥–∞—Ç Order –¥–æ–ª–∂–µ–Ω —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞—Ç—å OrderCreated –≤–æ –≤—Ä–µ–º—è —Å–≤–æ–µ–≥–æ —Å–æ–∑–¥–∞–Ω–∏—è, –∞ —Ç–∞–∫–∂–µ —Å–æ–±—ã—Ç–∏—è –≤–∏–¥–∞ Order* –ø—Ä–∏ –∫–∞–∂–¥–æ–º —Å–≤–æ–µ–º –æ–±–Ω–æ–≤–ª–µ–Ω–∏–∏. –≠—Ç–æ —Ç—Ä–µ–±–æ–≤–∞–Ω–∏–µ –∫—É–¥–∞ –±–æ–ª–µ–µ –∂–µ—Å—Ç–∫–æ–µ, —á–µ–º —Ç–æ, —á—Ç–æ –º—ã –≤–∏–¥–µ–ª–∏ —Ä–∞–Ω–µ–µ, –∫–æ–≥–¥–∞ –∞–≥—Ä–µ–≥–∞—Ç –≥–µ–Ω–µ¬≠—Ä–∏—Ä–æ–≤–∞–ª —Ç–æ–ª—å–∫–æ —Ç–µ —Å–æ–±—ã—Ç–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –±—ã–ª–∏ –∏–Ω—Ç–µ—Ä–µ—Å–Ω—ã –ø–æ—Ç—Ä–µ–±–∏—Ç–µ–ª—è–º. –ù–æ —ç—Ç–æ –µ—â–µ –Ω–µ –≤—Å–µ. –°–æ–±—ã—Ç–∏–µ –¥–æ–ª–∂–Ω–æ —Å–æ–¥–µ—Ä–∂–∞—Ç—å –¥–∞–Ω–Ω—ã–µ, –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã–µ –∞–≥—Ä–µ–≥–∞—Ç—É –¥–ª—è –ø–µ—Ä–µ—Ö–æ–¥–∞ –∫ –Ω–æ–≤–æ–º—É —Å–æ—Å—Ç–æ—è–Ω–∏—é. –ù–µ–∫–æ—Ç–æ—Ä—ã–µ —Å–æ–±—ã—Ç–∏—è, —Ç–∞–∫–∏–µ –∫–∞–∫ Order Shipped, —Å–æ–¥–µ—Ä–∂–∞—Ç –Ω–µ–º–Ω–æ–≥–æ –¥–∞–Ω–Ω—ã—Ö (–∏–ª–∏ –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—Ç –Ω–∏–∫–∞–∫–∏—Ö) –∏ –ø—Ä–æ—Å—Ç–æ –æ–ø–∏—Å—ã–≤–∞—é—Ç –ø–µ—Ä–µ—Ö–æ–¥ —Å–æ—Å—Ç–æ—è–Ω–∏—è. –ú–µ—Ç–æ–¥ apply() –æ–±¬≠—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç —Å–æ–±—ã—Ç–∏–µ Order Shipped, –º–µ–Ω—è—è –ø–æ–ª–µ –∑–∞–∫–∞–∑–∞ status –Ω–∞ SHIPPED.

### **–ú–µ—Ç–æ–¥—ã –∞–≥—Ä–µ–≥–∞—Ç–∞ –ø–æ–ª–Ω–æ—Å—Ç—å—é –ø–æ–ª–∞–≥–∞—é—Ç—Å—è –Ω–∞ —Å–æ–±—ã—Ç–∏—è**
